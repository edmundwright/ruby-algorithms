## Time complexity of operations

|            | Time best case | Time worst case | Time average case |
|------------|----------------|-----------------|-------------------|
|`#include?` | `O(1)`         | `O(n)`          | `O(1)`            |
|`#insert`   | `O(1)`         | `O(n)`          | `O(1)` (amortized)|
|`#delete`   | `O(1)`         | `O(n)`          | `O(1)`            |

## Memory

`O(n)`
